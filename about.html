A Little About Myself

I began teaching myself to code as a hobby after work only to discover that I loved it. Shortly after, my new career path started.

I find the puzzles in this line of work to be interesting and quite exciting. The challenges of learning new languages and fixing issues deep in the code is what I thrive on. Programming Javascript interactivity with vanilla JS is my current passion.

At the present moment, I have focused mainly on front-end development, and have honed my skills in that area. Next I will be studying full-stack Javascript development as well as looking into Python. As a Linux enthusiast, I work in a modified Ubuntu desktop environment, but am confident with both Windows and Mac. A constant learner, I am continually expanding my skillset.

Skills

Utilizing online tools such as Team Treehouse and Free Code Camp, I have gained around a years' experience in web design and have completed projects using the following technology:

  • HTML

    Confident in the use of semantic markup to produce easy to read code and ease of use for screen readers.

  • CSS

    Even with frameworks like Bootstrap, CSS is my bread and butter for creating interesting responsive web environments.

  • Sass

    Sass is an indispensible tool for organizing and simplifying CSS. I use it on almost every project I do.

  • JavaScript

    JavaScript is the single most important thing I have learned. It has opened me up to the exciting world of possibilities in web design and beyond. Solving JavaScript puzzles has become a passion of mine.

  • Jquery

    Jquery is very useful for simplifying JavaScript element selection, adding effects to a website, or simply adding a plugin. I am familiar with Jquery and use it for adding certain plugins, but overall prefer vanilla Javascript for most of my DOM traversal and manipulation.

  • Bootstrap

    I am familiar with Bootstrap basics and hope to learn more in the near future.

  • Linux

    The use and customization of Linux desktop environments is a hobby of mine. I like to make my desktop environment visually appealing and functionally simplistic.
