A Little About Myself

I began teaching myself to code as a hobby after work only to discover that I loved it. Shortly after, my new career path started.

I find the puzzles in this line of work to be interesting and quite exciting. The challenges of learning new languages and fixing issues deep in the code is what I thrive on. React development is my current focus.

At the present moment, I am experienced and confident in front end development using mainly HTML5, Sass, and React. I am currently studying back end JavaScript (node and express) so a can eventually round out my skills to a full stack. As a Linux enthusiast, I work in a modified Ubuntu desktop environment, but am confident with both Windows and Mac. A constant learner, I am continually expanding my skillset.

Skills

  • HTML

    Confident in the use of semantic markup to produce easy to read code and ease of use for screen readers.

  • CSS

    Even with frameworks like Bootstrap, basic CSS is my bread and butter for creating interesting responsive web environments.

  • Sass

    Sass is an indispensible tool for organizing and simplifying CSS. I use it on almost every project I do.

  • JavaScript

    JavaScript is the single most important thing I have learned. It has opened me up to the exciting world of possibilities in web design and beyond. Solving JavaScript puzzles has become a passion of mine.

  • React.js

    I love using React to create modular, component-based websites and user interfaces. I will be converting many of my projects over to React.

  • Bootstrap

    I use bootstrap when a page's layout needs to be designed quickly.

  • Linux

    The use and customization of Linux desktop environments is a hobby of mine. I like to make my desktop environment visually appealing and functionally simplistic.
